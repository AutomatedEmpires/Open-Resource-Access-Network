# Agents

This folder contains agent-facing docs: activation context, status snapshots, prompts, and the agent roadmap.

## Key docs

- `AGENTS_OVERVIEW.md` — what the agents are and where they operate
- `ROADMAP.md` — build plan / roadmap

## Subfolders

- `activation/` — per-agent activation docs
- `status/` — status snapshots (point-in-time)
- `prompts/` — prompt notes/playbooks
